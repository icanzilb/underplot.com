Description

dataTile for Simulator is an innovative Xcode companion-app to see debugging data in a beautiful and bold UI. Developers log debug data as usual using Apple’s logging framework, and dataTile for Simulator automatically selects and displays the logs!

It’s a powerful native developer tool that is focused on making developers more productive while developing with Xcode and the Simulator. There is no need for third-party frameworks, hefty configurations, or changes to the code if already using Apple’s unified logging in Swift or Objective-C.

dataTile for Simulator helps developers reduce development time, debug with ease and track state effortlessly by autonomously detecting data and displaying:

  • Text tile to show prominently text or numbers.
  • Chart tile to plot numerical data over time.
  • Switch tile to display boolean data.
  • Progress tile for progress data over time.
  • With more tiles on the way!

The app is free to download with minimal functionality and PRO features can be unlocked via purchasing an in-app subscription.

dataTile Mini for Simulator

The always-free dataTile Mini version of the app is ideal for working with just a few pieces of simple debug data for students and hobbyists who are learning iOS development.

dataTile Pro for Simulator

A generous free-trial period unlocks the PRO features from within the app to enjoy unlimited tiles, smart tiles to do more with your data, persisting tile-layout per app, more log recognition patterns, and more.
