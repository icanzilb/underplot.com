dataFude 1.10

  • Tile Press Release
  • November 22, 2023

dataFude 1.10 is an update unlocking all features to all users.

Currently active subscriptions will not renew at the next billing cycle if users do not cancel manually before the end of their respective billing cycle.

The Mini and Pro versions of the app become the same free tier version with all features unlocked.

The in-development MAX version of the app is cancelled.

Enjoy dataFude 1.10!
